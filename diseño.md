Quiero que diseñes una página web moderna, sencilla y elegante para un emprendimiento de manualidades llamado "Creaciones Rosa Elena".

El diseño debe tener un estilo femenino, minimalista y limpio, usando principalmente tonos rosados suaves, blanco y colores pastel.

Estructura requerida:

1. HEADER (encabezado)
- Barra superior con fondo rosado.
- Nombre del negocio alineado a la izquierda.
- Menú de navegación alineado a la derecha con enlaces: Inicio, Productos y Contacto.
- Tipografía elegante y fácil de leer.

2. SECCIÓN PRINCIPAL – PRODUCTOS
- Título centrado que diga "Productos".
- Mostrar categorías en tarjetas (cards) horizontales con bordes redondeados y sombra suave.
- Cada tarjeta debe incluir un ícono y el nombre del producto.
- Categorías:
  - Lazos
  - Piñatas
  - Birretes
  - Decoraciones
- Diseño responsive adaptable a móvil.

3. SECCIÓN DE CONTACTO
- Fondo rosado suave.
- Mostrar ubicación con ícono (ej: Valencia).
- Mostrar redes sociales (Instagram).
- Diseño centrado y limpio.

4. ESTILO VISUAL
- Bordes redondeados.
- Sombras suaves (soft shadows).
- Espaciado amplio y ordenado.
- Estética moderna tipo tienda artesanal.
- Uso de iconos pequeños y elegantes.

5. REQUISITOS TÉCNICOS
- HTML5 + CSS3.
- Diseño responsive.
- Código organizado y comentado.
- Usar Flexbox o Grid.
- Evitar frameworks pesados.

Genera el código completo listo para usar.
